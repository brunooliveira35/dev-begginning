Git & Github

Criado em 2005 por Linus Torvalds, o Git é um sistema essencial para programadores. Esse sistema tem a capacidade de registrar quaisquer alterações feitas em um código, armazenando essas informações e permitindo que o programador reverta para uma versão anterior, se necessário, enquanto o Github funciona como uma rede social para esses programadores onde se pode criar repositórios, hospedar e compartilhar seus projetos, seguir outros programadores ou colaborar com softwares open source.
