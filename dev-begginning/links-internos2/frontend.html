Front-End

Para o desenvolvimeto de programas, sistemas, sites e aplicativos, o Front-end é parte essencial do projeto. O front-end esta muito relacionado com a parte de interface gráfica de um projeto, portanto é essencial que o desenvolvedor se atente com o UX, pois o usuário irá interagir diretamente, assim fornecendo uma melhor experiência.

Caso um site, programa ou sistema não seja muito intuitivo ou simples de entender, o público perde o interesse. O papel de um desnvolvedor front-end é garantir que a ferramenta funcione conforme o esperado e facilitar a usabilidade. O front-end vai cuidar da aparência do que está sempre mudando para se adequar melhor ao usuário, a segurança e a usabilidade.

Existem várias ferramentas para se fazer projetos assim, nesse foi utilizado o Visual studio Code, bastante util para fazer o HTML e o CSS que serve para dar uma personalidade ao nosso site!
